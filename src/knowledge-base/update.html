Оновлення та зміни

Завантажити оновлення можна зі сторінки оновлень в програмі (меню Система).

У випадку проблем можна виконати ручне оновлення відповідно опису оновлення або виконати повне оновлення з готовї збірки

Відкотити зміни у разі проблем можна з архіву попередньої версії

Якщо пропущено кілька оновлень, потрібно виконати повне оновлення папок і файлів (крім config) завантаживши останню версію з GitHub або завантаживши готову збірку Також можливо треба буде послідовно виконати sql скрипти з папки db/updates.
