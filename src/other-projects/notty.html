Notty

Zippy Note – Менеджер персональных записей (блокнот, органайзер, база знаний), представляющий возможность сохранения и упорядочивания пользовательских записей в иерархическом виде.

Контент разделяется между пользователями, поэтому может работать как SAAS.

Проект возник в результате статьи на Хабре

Основной репозитарий на GitHub

Демо версия вход admin@admin.admin пароль admin

Пользователи, не являющиеся IT специалистами могут скачать "коробочную" версию - сайт упакован в архив вместе с веб-сервером и сервером БД (для windows 7 и выше).
